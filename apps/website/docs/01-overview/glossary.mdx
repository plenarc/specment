## 用語一覧 {#glossary-list}

:::tip TIP
1. 以下は記述例
1. すべての記述例を削除、もしくは編集したらこのTIPも削除する
:::

### API
1. Application Programming Interface の略。ソフトウェア間の通信仕様
1. REST, gRPC など

### Availability
1. システムの稼働率を示す指標
1. 「可用性」とも呼ぶ

### BPMN
1. Business Process Model and Notation。業務プロセス図の標準表記
1. PlantUML 等で描画可能  

### キャパシティプランニング {#capacity-planning}
1. 将来の負荷増加を見越した資源計画  
1. CPU / メモリ / IOPS など
